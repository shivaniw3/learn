Common CSS properties

Background

Cascading style sheet describe the appearance of structured documents,usually written in HTML.The current standard for CSS is level2 revision1(css2.1)which ih officially documented by W3C in the Cascading Style Sheet Level2 Revision1(CSS2.1) Specification.CSS is the new emerging standard.A good website is follow what is going on with CSS3 is www.css3.info.

color #RRGGBB(Red Green Blue he values) any element that contains text

text transformation none|capitalize|upercase|lowercase any element that contains text
